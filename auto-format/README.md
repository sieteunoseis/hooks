# auto-format

Auto-format files after Claude writes or edits them.

## Install

```bash
npx @cmds-cc/hooks add sieteunoseis/hooks/auto-format
```

## Hooks

| Hook                   | Default | Description                                                       |
| ---------------------- | ------- | ----------------------------------------------------------------- |
| Prettier on write/edit | On      | Runs prettier --write on files after Claude creates or edits them |
