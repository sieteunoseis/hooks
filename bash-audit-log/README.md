# bash-audit-log

Log all bash commands Claude runs to an audit file.

## Install

```bash
npx @cmds-cc/hooks add sieteunoseis/hooks/bash-audit-log
```

## Hooks

| Hook              | Default | Description                                                          |
| ----------------- | ------- | -------------------------------------------------------------------- |
| Log bash commands | On      | Appends every bash command to ~/.claude/bash-log.txt with timestamps |
